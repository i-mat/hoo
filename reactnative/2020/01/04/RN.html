04_reactnative布局

Andy Sun 04 Jan 2020

React Native 中布局采用的是FlexBox(弹性框)进行布局。

FlexBox提供了在不同尺寸设备上都能保持一致的布局方式。FlexBox是CSS3弹性框布局规范，目前还处于最终征求意见稿阶段，并不是所有的浏览器都支持FlexBox。

宽和高

一个组件的高度和宽度决定了它在屏幕上的尺寸，也就是大小。

像素无关

在rn中尺寸是没有单位的，它代表了设备独立像素。

这里要注意reactnative中组件样式style，应该是双括号的形式，但是这个和jekyll中liquid语法冲突，liquid使用双括号来引用变量使用的，所以从此以后，文档中关于style后内容都用单括号表示。 
<View style={width:100,height:100,margin:40}>
	<Text style={fontSize:16,margin:20}></Text>
</View>

上述代码中，运行在android上时，View的长和宽被解释成：100dp 、 100dp，单位是dp，字体被解释成16sp，单位是sp 。

运行在ios上尺寸单位被解释成pt，这些单位确保了布局在不同dpi的手机屏幕上显示不会发生改变。

和而不同

rn中FlexBox和web css中的FlexBox工作方式一样，但是有细微区别，如：

  • flexDirection：rn中默认 column, web中默认row
  • alignItems: rn默认 stretch(拉伸) ,web默认 flex-start
  • flex： rn 只接受一个参数，web中可接受多个 ，如 flex: 2 2 10%
  • rn不支持 align-content, flex-basis, order, flex-basis ,flex-flow , flex-grow , flex-shrink

Layout Props

以下属性是rn所支持的flex属性

父视图属性(容器属性):

  • flexDirection enum (‘row’,’column’,’row-reverse’,’column-reverse’)
  • flexWrap enum(‘wrap’,’nowrap’)
  • justifyContent enum (‘flex-start’,’flex-end’,’center’,’space-between’,’space-around’)
  • alignItems enum(‘flex-start’,’flex-end’,’center’,’stretch’)

主轴和侧轴（横轴和竖轴）

在学习上述属性之前，让我们先了解一个概念： 主轴和侧轴

主轴即水平方向的轴线，可以理解成横轴，侧轴垂直于主轴，可以理解为竖轴。

flexDirection

enum (‘row’,’column’,’row-reverse’,’column-reverse’)

flexDirection 属性定义了父视图中的子元素沿横轴或竖轴方向的排列方式。

  • row： 从左向右依次排列
  • row-reverse: 从右向左依次排列
  • column(default)：默认的排列方式，从上向下排列
  • column-reverse: 从下向上排列
<View style={flexDirection: 'row-reverse',backgroundColor:'darkgrey',marginTop:20}>
  <View style={width:40,height:40,backgroundColor:'darkcyan',margin:5}>
  	<Text style={fontSize:16}>1</Text>
  </View>
  <View style={width:40,height:40,backgroundColor:'darkcyan',margin:5}>
  	<Text style={fontSize:16}>2</Text>
  </View>
  <View style={width:40,height:40,backgroundColor:'darkcyan',margin:5}>
  	<Text style={fontSize:16}>3</Text>
  </View>
  <View style={width:40,height:40,backgroundColor:'darkcyan',margin:5}>
  	<Text style={fontSize:16}>4</Text>
  </View>
</View>

flexWrap

enum(‘wrap’,’nowrap’)

flexWrap属性定义了子元素在父视图内是否允许多行排列，默认为nowrap

  • nowrap flex的元素值排列在一行上，可能导致溢出。
  • Wrap flex的元素在一行排列不开的时候，就进行多行排列
<View style={flexWrap:'wrap',flexDirection:'row'}>...</View>

justifyContent

enum (‘flex-start’,’flex-end’,’center’,’space-between’,’space-around’)

justifyContent属性定义了浏览器如何分配顺着父容器主轴的弹性(flex)元素之间及其周围的空间，默认为flex-start

  • flex-start(default) 从行首开始排列，每行第一个弹性元素与行首对齐，同时所有后续的弹性元素与前一个对齐
  • flex-end 从行尾开始排列。
  • center 伸缩元素向每行中点排列。每行第一个元素到行首的距离与最后一个元素到行尾的距离相同。
  • space-between 在每行上均匀分配弹性元素。相邻元素距离相同。每行第一个元素与行首对齐，最后一个与行尾对齐。
  • space-around 在每行上均匀分配弹性元素，相邻元素间距离相同。每行第一个元素到行首的距离和每行最后一元素到行尾的距离将会是相邻元素之间距离的一半。

alignItems

enum(‘flex-start’,’flex-end’,’center’,’stretch’)

alignItems属性与justifyContent相同的方式在侧轴方向上将当前行上的弹性元素对齐，默认为stretch.

  • flex-start 元素向侧轴起点对齐
  • flex-end 终点对齐
  • center 元素在侧轴居中。如果元素在侧轴上的高度高于其容器，那么再两个方向上溢出距离相同
  • stretch 弹性元素被在侧轴方向上被拉伸到与容器相同的高度或宽度。

子视图属性

  • alignSelf enum(‘auto’, ‘flex-start’, ‘flex-end’, ‘center’, ‘stretch’)
  • flex number

alignSelf

alignSelf属性定义了flex容器内被选中项目的对齐方式。注意：alignSelf属性可重写灵活容器的alignItems属性。

  • auto(default) 元素继承了它的父容器的alignItems属性，如果没有父容器则为 stretch
  • stretch 元素被拉伸以适应容器
  • center 元素位于容器的中心
  • flex-start 元素位于容器的开头
  • flex-end 元素位于容器的末尾

flex

flex 属性定义了一个可伸缩元素的能力，默认为0，可以视为兄弟节点的权重比

其他布局 in rn

视图边框

  • borderBottomWidth number 底部边框宽度
  • borderLeftWidth 左边框
  • borderRightWidth 右边框
  • borderTopWidth 顶部边框
  • borderWidth 边框宽度

尺寸

  • width
  • height

外边距

  • margin 外边距
  • marginBottom 底部外边距
  • marginHorizontal 左右外边距
  • marginLeft 左外边距
  • marginRight
  • marginTop
  • marginVertical 上下外边距

内边距

  • padding
  • paddingBottom
  • paddingHorizontal
  • ..

边缘

  • left 规定元素的左边缘。该属性定义了定位元素做外边距边界与父模块左边界之间的偏移
  • right
  • top
  • bottom

定位(position)

enum(‘absolute’,’relative’) 设置元素的定位方式，为将要定位的元素定义定位规则

  • absolute 生成绝对定位的元素，元素的位置通过 left,right,top,bottom属性进行规定
  • relative 生成相对定位的元素，相对于其正常位置进行定位.因此，”left:20”会向元素的left位置添加20像素
