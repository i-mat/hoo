2,初始化

c语言不像java一样，数据类型都有默认初始值，所以c语言中，需要在程序开头，先将变量初始化。

C99 和 ANSI C 区别

  1. C99中可在任意地方定义变量；而传统的ANSI C 只能在代码开头的地方定义变量。

  2. 变量是可变的量，常量是固定不变的量，直接写在程序里的，我们称作直接量（literal），在程序中多次出现的直接量，是magic number不利于阅读与维护；

C99允许直接定义常量 ，在类型前边加修饰符 const，这个值不能被修改，read-only variable is not assignable。

ex: const int A=100; （注意与宏定义相区别）

  1. 单行注释 // 是C99独有的，ANSI C不支持；而多行注释 /* */是传统ANSI C就有的

  2. for(int i=0;i<n;i++)这种在for循环里定义变量是C99独有的

  3. C99在定义数组时，数组的个数可以是变量

6.C99进行数组的集成初始化时，可定位赋值。

ex: int a[10]={[0]=20,[5]=,5,6};

输出：20，0,0,0,0,5,6,0,0,0

3,读整数

int price = 0;

scanf(“%d”, &price);

要求scanf 这个函数读入下一个整数，读到的结果赋值给变量price

4,常量

const int AMOUNT = 100;

常量 建议 都大写

const是修饰符，表示这个变量的值一旦初始化之后，就不能再修改。
